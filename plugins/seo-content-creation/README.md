# SEO Content Creation

> Agents for planning, writing, and auditing SEO-optimized content with E-E-A-T best practices.

## Overview

This plugin provides a complete content creation pipeline for search engine optimization. It covers the full lifecycle from strategic content planning and topic cluster design, through writing comprehensive SEO-optimized articles, to auditing existing content for quality and optimization opportunities. Use this plugin when producing content intended to rank in search engines or when improving existing content's search visibility.

## Contents

### Agents

- **seo-content-planner** -- Creates comprehensive content outlines, topic clusters, content calendars, and identifies content gaps. Maps search intent and builds pillar content strategies. Runs on Haiku for fast, lightweight planning.

- **seo-content-writer** -- Writes SEO-optimized articles with natural keyword integration, E-E-A-T signals, and compelling structure. Produces full content packages including title variations, meta descriptions, and FAQ sections. Runs on Sonnet for high-quality output.

- **seo-content-auditor** -- Analyzes existing content for quality, E-E-A-T signals, readability, keyword usage, and trust indicators. Produces scored audit reports with actionable improvement recommendations. Runs on Sonnet for thorough analysis.

## Usage

Install this plugin into your Claude Code project:

```bash
claude plugin install seo-content-creation
```

Then use the agents by delegating SEO content tasks. For example, ask the content planner to build a topic cluster, have the writer produce articles from those outlines, and run the auditor against your drafts for optimization feedback.
