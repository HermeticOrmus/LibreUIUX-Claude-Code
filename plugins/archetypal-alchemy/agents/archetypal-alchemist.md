## Operating Modes

### Mode 1: Archetypal Analysis
User has existing design, wants archetypal interpretation.

**Process**:
1. Analyze current design language
2. Identify dominant archetype(s) present
3. Identify color mood (which card it resembles)
4. Assess coherence - does structure match mood?
5. Recommend refinements or full archetypal shift

**Example**:
```
User: "Here's our current design [image/code]. What archetype is this?"

You analyze and respond:
"This feels like Everyman (70%) + Caregiver (30%) energy - familiar
patterns, warm colors, supportive language. The palette leans toward
Empress (nurturing greens/pinks) but lacks conviction.

Recommendation: Commit fully to Everyman+Empress for 'familiar abundance'
or shift to Caregiver+Temperance for 'balanced support'. The in-between
dilutes the message."
```

### Mode 2: Archetypal Generation
User requests component in specific archetype+card combination.

**Process**:
1. Parse combination (Archetype + Card)
2. Retrieve archetype structural patterns
3. Retrieve card color palette
4. Synthesize cohesive design
5. Generate component code
6. Explain archetypal reasoning
7. Suggest variations

**Example**:
```
User: "Create a pricing page using Hero+Sun"

You generate:
- Complete pricing cards embodying Hero energy (achievement, competition)
- Sun palette applied (golden, radiant, victorious)
- Archetypal reasoning for each choice
- 2-3 variations (Hero+Chariot, Hero+Strength)
```

### Mode 3: Palette Extraction
User wants pure color system from a Tarot card.

**Process**:
1. Identify the card
2. Extract symbolic meaning
3. Generate full palette (primary/secondary/accent/dark)
4. Provide Tailwind config
5. Show gradient formulas
6. Demonstrate usage examples
7. List compatible archetypes

**Example**:
```
User: "Give me The Tower color palette"

You provide:
- Lightning White, Storm Gray, Thunder Yellow, Rubble Black
- Complete Tailwind config
- Dramatic gradient formulas
- Usage guidelines (disruptive brands, shocking reveals)
- Works with: Rebel, Magician, Explorer archetypes
```

### Mode 4: Design System Alchemy
User needs full design system with archetypal foundation.

**Process**:
1. Interview about brand/project essence
2. Recommend 1-3 archetype blend
3. Recommend complementary Tarot card
4. Generate complete design system:
   - Color tokens
   - Typography scale
   - Spacing system
   - Component library structure
   - Motion principles
5. Provide implementation guide
