# Content Marketing

> Agents for AI-powered content creation, SEO optimization, omnichannel distribution, and deep web research.

## Overview

The Content Marketing plugin provides specialized agents for developing and executing data-driven content marketing strategies. It pairs a content marketing strategist with a search specialist to cover the full content lifecycle: research, creation, optimization, distribution, and performance analysis. Use this plugin when building content strategies, creating SEO-optimized content, running competitive analysis, or establishing thought leadership across digital channels.

## Contents

### Agents

- **content-marketer** -- Elite content marketing strategist specializing in AI-powered content creation, omnichannel distribution, and conversion optimization. Integrates with modern AI writing tools (Agility Writer, ContentBot, Jasper) for automated content workflows and bulk generation. Covers SEO-optimized content with real-time SERP data, topical mapping, content cluster development, and compliance with Google's Helpful Content guidelines.

- **search-specialist** -- Expert web researcher using advanced search techniques and multi-source synthesis. Masters search operator formulation, domain-specific filtering, result quality evaluation, information synthesis across sources, fact verification, cross-referencing, and historical trend analysis. Supports competitive analysis and deep research workflows.

### Commands

This plugin contains agents only -- no standalone commands. The agents activate automatically when content marketing or research tasks are detected.

### Skills

This plugin contains no skills.

## Usage

```
# Install to your project
cp -r plugins/content-marketing .claude/plugins/

# The agents activate automatically for content-related tasks
# Example prompts:
# "Create a content strategy for our developer blog"
# "Research competitors in the API monitoring space"
# "Write an SEO-optimized article about microservices patterns"
```
