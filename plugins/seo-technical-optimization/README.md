# SEO Technical Optimization

> Agents for keyword strategy, meta tag optimization, featured snippet targeting, and content structure architecture.

## Overview

This plugin focuses on the technical side of search engine optimization -- the mechanics that make content discoverable and well-presented in search results. It provides specialized agents for keyword density analysis, metadata crafting, featured snippet formatting, and header hierarchy optimization with schema markup. Use this plugin when you need to optimize how content is structured, tagged, and formatted for search engines.

## Contents

### Agents

- **seo-keyword-strategist** -- Analyzes keyword usage in content, calculates density, generates LSI and semantic keyword suggestions, maps entities, and detects over-optimization. Runs on Haiku for rapid analysis.

- **seo-meta-optimizer** -- Creates optimized meta titles, descriptions, and URL suggestions within character/pixel limits. Generates multiple A/B test variations with emotional triggers and CTAs. Supports WordPress and static site platforms. Runs on Haiku.

- **seo-snippet-hunter** -- Formats content blocks for featured snippet eligibility (paragraph, list, and table formats). Optimizes question-answer structures and FAQ sections for "People Also Ask" dominance. Includes schema markup templates. Runs on Haiku.

- **seo-structure-architect** -- Analyzes and optimizes header hierarchy (H1-H6), suggests schema markup (Article, FAQ, HowTo, BreadcrumbList), builds internal linking matrices, and designs content silo strategies. Runs on Haiku.

## Usage

Install this plugin into your Claude Code project:

```bash
claude plugin install seo-technical-optimization
```

Use the agents proactively when creating or optimizing web content. For example, run the keyword strategist on a draft to check density, then use the meta optimizer to generate title tags and descriptions, the snippet hunter to format FAQ sections, and the structure architect to validate your heading hierarchy and generate schema markup.
